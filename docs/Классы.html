Классы:

Planet

Класс планеты. Имеет следующие поля:

  • name:string -- имя планеты;
  • typ:string -- тип планеты (название type является ключевым словом python3);
  • coordinates:(int, int) -- координаты планеты в декартовой системе координат;

Класс на данный момент не доработан, потому что не продуманы свойства планеты.

Star

Класс, описывающий звезду. Имеет следующие поля:

  • name:string -- имя звезды;
  • planetsNumber:int -- кол-во планет, принадлежащих звезде;
  • neighbours:(int, int) -- смежные с оной звёзды: первое число -- длина пути, второе -- номер звезды, к которой путь ведёт, в графе; 

Класс на данный момент не доработан, потому что не продуманы свойства звезды.
