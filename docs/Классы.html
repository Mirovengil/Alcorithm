Классы:

Planet

Абстракный класс планеты. Имеет следующие поля:

  • name : string -- название планеты;
  • coordinates : (int, int) -- координаты планеты на планетарной карте;
  • type : string -- тип планеты;
  • steel : float -- коэффициент, на который домножается количество добываемой на планете стали;
  • food : float -- коэффициент, на который домножается количество добываемой на планете еды;
  • money : float -- коэффициент, на который домножается количество добываемых на планете денег;
  • description : string -- текстоое описание планеты, которое видит игрок;
  • port : [Port] -- массив имеющихся на планете космических портов (см. соответствующий класс Port);
  • defence : [Defence] -- массив имеющихся на планете планетарных гарнизонов (см. соответствующий класс Defence);
  • fun : [Fun] -- массив имеющихся на планете развлекательных сооружений (см. соответствующий класс Fun);
  • farm : [Farm] -- массив имеющихся на планете ферм (см. соответствующий класс Farm);
  • main : [Main] -- массив имеющихся на планете шахт (см. соответствующий класс Main)

Важно: т.к. класс является абстрактным, то не рекомендуется создавать объекты его типа. Поля коэффициентов будут иметь значение None. Предпочтительнее использовать классы конкретизированных планет, которые описаны ниже:

  • Lave -- лавовая планета (см. таблицу планет в документации);
  • Water -- водяная планета (см. таблицу планет в документации);
  • Ice -- ледяная планета (см. таблицу планет в документации);
  • Desert -- пустынная планета (см. таблицу планет в документации);
  • Earth -- планета Земного типа (см. таблицу планет в документации);
  • Air -- газовая планета (см. таблицу планет в документации);
  • Rock -- скалистая планета (см. таблицу планет в документации);

Все они наследуют класс Planet, переопределяя конструктор и некоторые методы из описанных ниже (см. ограничения на кол-во построек для планет).

Методы класса Planet:

  • __init__(name, coordinates) -- конструктор, на вход получает значения одноимённых полей;
  • getSteel() : int -- геттер, возвращает значение поля steel;
  • getFood() : int -- геттер, возвращает значение поля food;
  • getMoney() : int -- геттер, возвращает значение поля money;
  • getDescription() : string -- геттер, возвращает значение поля description;
  • getPort() : [Port] -- геттер, возвращает значение поля port;
  • getDefence() : [Defence] -- геттер, возвращает значение поля defence;
  • getFun() : [Fun] -- геттер, возвращает значение поля fun;
  • getFarm() : [Farm] -- геттер, возвращает значение поля farm;
  • getMain() : [Main] -- геттер, возвращает значение поля main;
  • getType() : string -- геттер, возвращает значение поля type;
  • getName() : string -- геттер, возвращает значение поля name;
  • getCoordinates() : (int, int) -- геттер, возвращает значение поля coordinates;
  • getBuildsLen() : int -- типа геттер, возвращает суммарное количество строений на планете;
  • addPort(port) : bool -- пытается построить порт, возвращает True (удалось) / False (нельзя); в случае успеха добавляет port в поле port;
  • addDefence(defence) : bool -- пытается построить планетарный гарнизон, возвращает True (удалось) / False (нельзя); в случае успеха добавляет defence в поле defence;
  • addFun(fun) : bool -- пытается построить развлекательное сооружение, возвращает True и/ False (нельзя); в случае успеха добавляет fun в поле fun;
  • addFarm(farm) : bool -- пытается построить ферму, возвращает True (удалось) / False (нельзя); в случае успеха добавляет farm в поле farm;
  • addMain(main) : bool -- пытается построить шахту, возвращает True (удалось) / False (нельзя); в случае успеха добавляет main в поле main;

Star

Класс, описывающий звезду. Имеет следующие поля:

  • name : string -- имя звезды;
  • planetsNumber : int -- кол-во планет, принадлежащих звезде;
  • neighbours : (int, int) -- смежные с оной звёзды: первое число -- длина пути, второе -- номер звезды, к которой путь ведёт, в графе; 

Класс на данный момент не доработан, потому что не продуманы свойства звезды.
